**ChatGPT API?**

Technically **[ChatGPT](https://chat.openai.com/) is a web app** that uses the **[Open AI Chat Completion API](https://platform.openai.com/docs/api-reference/chat/create)**.

- ChatGPT doesn't have it's own API
- Buuuut, **ChatGPT API** is easier to say and remember.

## PreReqs

To get started with the ChatGPT API you'll need to:

1. [Create an Open AI account](https://platform.openai.com)
2. Add some payment info
3. [Generate an API key](https://platform.openai.com/account/api-keys)
